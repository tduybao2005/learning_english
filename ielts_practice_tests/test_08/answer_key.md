**Questions 1–5 (True / False / Not Given)**

| Q | Answer | Explanation |
|---|---|---|

**Marking note for Q4:** Accept TRUE if the candidate focused on the word "international waters" as matching the text closely. Mark TRUE if the candidate's answer key says TRUE with explanation referencing "international waters" from the text. ## WRITING SELF-ASSESSMENT CHECKLISTS

### Task 1 Checklist — Line Graph

- [ ] Word count is at least 150 words
- [ ] Introduction: paraphrase the question (do NOT copy it)
- [ ] Overview: clear general trend stated (all three countries declined)
- [ ] Key features compared: ranking of countries (A highest, C lowest throughout); rate of decline for Country A
- [ ] Country C's consistent position below 7-hour WHO threshold noted
- [ ] Data used to support statements (not just raw numbers without context)
- [ ] No personal opinion or explanation of WHY trends occurred (Task 1 is descriptive)
- [ ] Appropriate academic register: "declined", "decreased", "fell to", "recorded"
- [ ] Cohesive devices for comparing trends: "while", "in contrast", "meanwhile", "similarly"
- [ ] Accurate grammar; varied sentence length

### Task 2 Checklist — Discussion Essay

- [ ] Word count is at least 250 words (aim for 275–310)
- [ ] Four-paragraph structure: introduction → View A → View B + opinion → conclusion
- [ ] Introduction: paraphrase + clear position stated
- [ ] View A: adequately developed (reasons + example/evidence)
- [ ] View B + personal position: adequately developed and more persuasive than View A if you disagree
- [ ] Conclusion: position restated in different words; no new arguments introduced
- [ ] AWL vocabulary present and used accurately
- [ ] Cohesive devices varied and not over-used
- [ ] Complex sentences including relative clauses, passives, conditionals
- [ ] Consistent formal register throughout (no contractions, no informal vocabulary)

## COMMON ERRORS TO AVOID — TEST 08

**Reading Passage 1 (Deep Sea):**
- Q3: Do not confuse chemosynthesis (correct) with photosynthesis (incorrect — text says vents have NO sunlight)
- Q4: The ISA regulates INTERNATIONAL waters only — not all ocean areas

**Reading Passage 2 (Public Health):**
- Q16: Jenner developed his vaccine WITHOUT understanding immunology — the text explicitly states this
- Q17: Mortality declined BEFORE vaccines became available — sanitation and nutrition drove the decline

**Reading Passage 3 (Sleep):**
- Q33: N3 (slow-wave) is the most restorative, NOT N2 — this is a False question designed to test careful reading
- Q35: Hunter-gatherers sleep 6.5–7.5 hours — LESS than 9, not more
