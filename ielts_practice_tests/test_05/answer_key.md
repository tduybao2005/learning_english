## PHẦN VIẾT (WRITING)

Xem model answers đầy đủ trong file **writing.md**.

### Tự đánh giá Task 1 — Quy trình xử lý nước thải

**Tiêu chí Task Achievement:**
- [ ] Mô tả đủ TẤT CẢ các giai đoạn (9 stages)?
- [ ] Viết câu tổng quan (overview) về toàn bộ quy trình?
- [ ] Đề cập đến điểm kết thúc (2 nhánh sử dụng)?
- [ ] Đạt tối thiểu 150 từ?

**Tiêu chí Ngôn ngữ:**
- [ ] Sử dụng thì bị động (passive voice) đúng cách: *water is collected / particles are removed / chlorine is used*?
- [ ] Sử dụng từ nối trình tự: *initially, subsequently, following this, at the next stage, finally*?
- [ ] Sử dụng từ vựng kỹ thuật phù hợp: *screening, sedimentation, aeration, filtration, disinfection, purification*?
- [ ] Không dùng liệt kê đơn giản mà có cấu trúc câu rõ ràng?

---

### Tự đánh giá Task 2 — Thành thị vs Nông thôn

**Tiêu chí Task Achievement:**
- [ ] Thảo luận CẢ HAI quan điểm (không bỏ sót một bên)?
- [ ] Nêu rõ ý kiến cá nhân và duy trì xuyên suốt?
- [ ] Mỗi luận điểm có dẫn chứng hoặc giải thích cụ thể?
- [ ] Đạt tối thiểu 250 từ?

**Tiêu chí Ngôn ngữ:**
- [ ] Sử dụng từ nối: *Furthermore / Nevertheless / Admittedly / In conclusion*?
- [ ] Từ vựng học thuật: *proponents, cultivate, unparalleled, mitigate, correlates*?
- [ ] Cấu trúc câu phức: *While... / Although... / Research suggests that...*?
- [ ] Không lặp lại từ ngữ trong câu hỏi (paraphrase introduction)?

## Ghi chú học tập — Test 05

**Điểm mạnh cần phát huy:**
- Nếu bạn xử lý tốt passive voice trong Task 1 → tiếp tục áp dụng cho các đề process diagrams khác
- Nếu bạn viết được essay discuss both views → thử loại agree/disagree và problem-solution tiếp theo

**Điểm cần cải thiện thường gặp:**
- Task 1 Process: Hay quên câu overview — luôn viết câu tổng quan ở đoạn 1
- Task 2: Hay viết quá nhiều cho một luận điểm, quên luận điểm còn lại
- Speaking Part 3: Trả lời quá ngắn — luyện thêm kỹ năng phát triển ý kiến

**Mục tiêu điểm Test 05:**
- Reading: ≥ 30/40 câu đúng (Band 7.0)
- Writing Task 1: Band 6.5–7.0
- Writing Task 2: Band 6.5–7.0
- Speaking: Band 6.5–7.0
