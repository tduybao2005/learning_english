### Passage 1: Marine Biology and Coral Reef Ecosystems (Q1–13)

---

**Q1–6: True / False / Not Given**

| Q | Answer | Explanation |
|---|---|---|
| 1 | TRUE | Paragraph A states directly: "covering less than one percent of the ocean floor" |
| 2 | FALSE | Paragraph B states zooxanthellae provide "up to 90 percent" of coral's energy — not all of it |
| 3 | TRUE | Paragraph A describes the Great Barrier Reef as "the largest living structure on the planet" |
| 4 | FALSE | Paragraph A specifies corals are found in "warm, shallow tropical waters" — not deep ocean |
| 5 | FALSE | Paragraph D states that rising temperatures (thermal stress) cause bleaching and expulsion of zooxanthellae; ocean acidification instead "weakens coral skeletons" — these are two separate threats |
| 6 | TRUE | Paragraph E states: "scientists consistently emphasise that effective long-term reef conservation is ultimately contingent upon significant reductions in global greenhouse gas emissions" |

**Q19–23: True / False / Not Given**

| Q | Answer | Explanation |
|---|---|---|
| 19 | FALSE | Paragraph B states Watt's engine "built upon the earlier atmospheric engine developed by Thomas Newcomen" — Newcomen's was earlier, so Watt's was not the first |
| 20 | TRUE | Paragraph D states Manchester's population grew from 25,000 to over 300,000 — described as "a twelve-fold increase", which is more than ten times |
| 21 | TRUE | Paragraph E: "the landmark report of the Sadler Committee in 1832" |
| 22 | FALSE | Paragraph F states the Chartist movement "advocated for broader political representation" — not primarily factory wages or conditions |
| 23 | NOT GIVEN | Paragraph E mentions the Factory Acts "mandated minimum ages for employment" but does not specify what those ages were; sixteen is never mentioned |

### Passage 3: The Neuroscience of Memory (Q27–40)

---

**Q27–30: Multiple Choice**

| Q | Answer | Explanation |
|---|---|---|
| 27 | B | Paragraph B: "procedural memories are largely implicit, expressed through performance rather than conscious recollection" |
| 28 | C | Paragraph C: following the removal of H.M.'s hippocampus, "H.M. was unable to form any new long-term declarative memories" — demonstrating the hippocampus is essential for this |
| 29 | B | Paragraph D: "During slow-wave sleep, the hippocampus repeatedly reactivates recently encoded memories, transferring information to the neocortex for long-term storage" |
| 30 | B | Paragraph E: Loftus showed that "leading questions… could alter participants' recollections" — i.e., the wording of a question changes what is remembered |

---

**Q31–36: True / False / Not Given**

| Q | Answer | Explanation |
|---|---|---|
| 31 | FALSE | Paragraph C states the hippocampus is located "in the medial temporal lobe of the brain" — not the frontal lobe |
| 32 | TRUE | Paragraph C: H.M.'s hippocampus was removed "to treat intractable epilepsy" — epilepsy causes severe, uncontrollable seizures |
| 33 | NOT GIVEN | Paragraph D describes both slow-wave and REM sleep roles but never compares their relative importance to each other |
| 34 | FALSE | Paragraph F: "approximately one quarter of participants came to develop… false memories" — not all participants |
| 35 | TRUE | Paragraph E states explicitly: "human memory is fundamentally reconstructive rather than reproductive" |
| 36 | NOT GIVEN | Paragraph G describes calls for reform from organisations like the American Psychological Association, but there is no claim that most countries have adopted these changes |

---

**Q37–40: Gap Fill**

| Q | Answer | Source in passage |
|---|---|---|
| 37 | sleep | Para D: "sleep plays a critical role in this consolidation process" |
| 38 | neocortex | Para D: "transferring information to the neocortex for long-term storage" |
| 39 | leading | Para E: "leading questions… could alter participants' recollections" |
| 40 | false | Para F: "participants… came to develop detailed, emotionally vivid false memories of this fictitious experience" |
