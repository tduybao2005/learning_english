## WRITING SELF-ASSESSMENT CHECKLISTS

### Task 1 Checklist

- [ ] Word count is at least 150 words
- [ ] Overview / general trend clearly stated (not just data listing)
- [ ] Most significant features selected and compared (not every data point)
- [ ] Data from both charts/graphs compared, not described separately
- [ ] Numbers included but not over-relied upon
- [ ] Appropriate academic register (no informal language)
- [ ] No personal opinion expressed (Task 1 is descriptive, not evaluative)
- [ ] Paragraphed logically: introduction → body → overview (or overview in introduction)
- [ ] Accurate grammar; varied sentence structures
- [ ] Wide vocabulary for comparing: "in contrast", "considerably more", "the most striking difference"

### Task 2 Checklist

- [ ] Word count is at least 250 words (aim for 270–310 for comfort)
- [ ] Task type correctly identified: "discuss both views AND give your opinion" = 4 paragraphs
- [ ] Introduction: paraphrase the question + state your position
- [ ] Body paragraph 1: View A — developed with reasons and/or examples
- [ ] Body paragraph 2: View B + your position — developed with reasons and/or examples
- [ ] Conclusion: restate position + summary (no new arguments)
- [ ] Consistent position maintained throughout
- [ ] AWL vocabulary used naturally (not forced)
- [ ] Complex sentence structures present (relative clauses, conditionals, passives)
- [ ] Cohesive devices used appropriately (not repetitively)
- [ ] Formal register maintained; no contractions, no slang
