## BÀI TẬP 1: Loại 0 hay Loại 1? — 20 phút

Chọn đúng thì và chia động từ:

1. If you _______ (heat) metal, it _______ (expand). (Type 0)
2. If you _______ (study) every day, your English _______ (improve) rapidly. (Type 1)
3. If the battery _______ (drop) below 10%, the robot _______ (return) to the charging station. (Type 1)
4. If water _______ (reach) 100°C, it _______ (boil). (Type 0)
5. If I _______ (pass) the IELTS exam, I _______ (apply) for the scholarship. (Type 1)
6. If the sensor _______ (detect) an obstacle, it _______ (trigger) an emergency stop. (Type 0/1)
7. If you _______ (press) Ctrl+C, the program _______ (stop). (Type 0)
8. If we _______ (finish) the prototype by Friday, we _______ (demonstrate) it next week. (Type 1)

**Đáp án:**
1. heat / expands | 2. study / will improve | 3. drops / will return | 4. reaches / boils | 5. pass / will apply | 6. detects / triggers (or will trigger) | 7. press / stops | 8. finish / will demonstrate

## BÀI TẬP 3: Unless — Chuyển câu — 15 phút

Chuyển từ "if...not" → "unless" hoặc ngược lại:

1. If you don't calibrate the sensors, the robot won't navigate correctly.
2. Unless you submit the report by Friday, the professor will deduct marks.
3. If you don't practice speaking, you won't improve your pronunciation.
4. Unless the system crashes, the test will be completed on time.
5. If you don't get enough sleep, you won't be able to focus during the exam.

**Đáp án:**
1. Unless you calibrate the sensors, the robot won't navigate correctly.
2. If you don't submit the report by Friday, the professor will deduct marks.
3. Unless you practice speaking, you won't improve your pronunciation.
4. If the system doesn't crash, the test will be completed on time.
5. Unless you get enough sleep, you won't be able to focus during the exam.

## BÀI TẬP 4: Dịch Việt → Anh — 20 phút

1. Nếu bạn luyện tập tiếng Anh mỗi ngày, bạn sẽ đạt 7.0 trong vòng một năm.
2. Trừ khi bạn hiệu chỉnh cảm biến, robot sẽ không thể điều hướng chính xác.
3. Nếu cảm biến phát hiện chướng ngại vật, hệ thống sẽ kích hoạt lệnh dừng khẩn cấp.
4. Nếu nhiệt độ vượt quá 80°C, CPU sẽ tự động tắt nguồn để bảo vệ.
5. Nếu tôi đậu kỳ thi IELTS lần này, tôi sẽ nộp đơn xin học bổng ngay.
6. Miễn là bạn tiếp tục cố gắng, bạn sẽ đạt được mục tiêu của mình.
7. Nếu bạn không backup code, bạn có thể mất tất cả công sức bỏ ra.
8. Nếu dự án thành công, trường sẽ hỗ trợ chúng tôi tham dự hội nghị quốc tế.

**Đáp án:**
1. If you practice English every day, you will achieve 7.0 within a year.
2. Unless you calibrate the sensors, the robot won't be able to navigate accurately.
3. If the sensor detects an obstacle, the system will trigger an emergency stop.
4. If the temperature exceeds 80°C, the CPU will automatically shut down to protect itself.
5. If I pass the IELTS exam this time, I will apply for a scholarship immediately.
6. As long as you keep trying, you will achieve your goals.
7. If you don't back up your code, you might lose all your hard work.
8. If the project succeeds, the university will support us to attend an international conference.

## PHẦN DICTATION — 15 phút

1. If you study English every day, you will definitely see improvement within a few months.
2. Unless the sensor is calibrated correctly, the robot won't navigate accurately.
3. If the battery drops below 20%, the system will automatically return to the charging station.
4. Provided that all tests pass successfully, we will deploy the robot next Monday.
5. If you don't back up your code before pushing, you might lose important changes.
6. As long as you remain consistent and focused, you will reach your IELTS target score.
7. If water reaches 100 degrees Celsius, it boils — this is a physical law, not a prediction.
8. What will you do if the navigation system fails during the hospital demonstration?
