# Nhật ký học Tiếng Anh - Ngày 34 (24/06/2026)
*Câu điều kiện Loại 1 — Conditionals Type 1 (Real/Possible Conditions)*

---

## PHẦN LÝ THUYẾT (30 phút)

### 1. Tổng quan các loại câu điều kiện

| Loại | Tên | Khả năng | Ví dụ |
|---|---|---|---|
| **Type 0** | Điều kiện chung / Khoa học | Luôn đúng | If you heat water, it boils. |
| **Type 1** | Điều kiện thực tế | Có thể xảy ra | If you study hard, you will pass. |
| **Type 2** | Điều kiện giả định | Khó xảy ra | If I were rich, I would buy a lab. |
| **Type 3** | Điều kiện không có thực | Không thể xảy ra | If I had studied, I would have passed. |

*Hôm nay tập trung: **Type 0 và Type 1***

### 2. Câu điều kiện Loại 0 (Zero Conditional)

**Cấu trúc:** If + Simple Present, Simple Present

*Dùng cho:* Sự thật chung, quy luật khoa học, hướng dẫn sử dụng
- If you heat ice, it **melts**.
- If the battery dies, the robot **stops** immediately.
- If you press this button, the system **restarts**.

### 3. Câu điều kiện Loại 1 (First Conditional)

**Cấu trúc:** If + Simple Present, will + V1

*Dùng cho:* Điều kiện thực tế có thể xảy ra trong tương lai

| Mệnh đề điều kiện (If-clause) | Mệnh đề kết quả (Main clause) |
|---|---|
| If + S + **V (Simple Present)** | S + **will** + V1 |
| If the test **passes**, | the robot **will** be deployed. |
| If you **practice** daily, | your English **will** improve. |

**Thay thế will trong mệnh đề kết quả:**
- If + Present... → **can** + V (khả năng): *If you help me, I **can** finish faster.*
- If + Present... → **may/might** + V (dự đoán ít chắc): *If we test it, it **might** work.*
- If + Present... → **should** + V (lời khuyên): *If you're tired, you **should** rest.*
- If + Present... → **must/have to** + V (nghĩa vụ): *If you find a bug, you **must** report it.*

### 4. Đảo vị trí mệnh đề

Có thể đặt mệnh đề kết quả trước:
- *You **will** improve **if** you **practice** daily.* (không dùng dấu phẩy)

*Chú ý:* Chỉ dùng dấu phẩy khi mệnh đề IF đứng **trước**:
- ✅ **If** you study, you **will** pass.
- ✅ You **will** pass **if** you study. ← không dấu phẩy

### 5. UNLESS = If...not
- *Unless you **calibrate** the sensor, the robot **won't** navigate correctly.*
= *If you **don't calibrate** the sensor, the robot **won't** navigate correctly.*

### 6. Hội thoại thực tế
> **A:** Do you think HospitGo will be ready for the hospital trial?
> **B:** If we fix the remaining sensor issues, it will definitely be ready.
> **A:** What will happen if the navigation fails during the trial?
> **B:** If it fails, the system will send an alert and stop immediately. We've programmed a safety protocol.
> **A:** Will you be there for the trial?
> **B:** Of course. If they allow observers, I'll attend. If not, I'll monitor remotely.
> **A:** What if the battery runs out mid-delivery?
> **B:** If the battery drops below 20%, the robot will automatically return to the charging station. Unless there's a critical delivery — in that case, it will complete the task first.
