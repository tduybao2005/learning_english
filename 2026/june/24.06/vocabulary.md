## 1. TỪ CHỈ ĐIỀU KIỆN

| Từ/Cụm | Nghĩa | Ví dụ |
|---|---|---|
| **if** | nếu | **If** you study hard, you will pass. |
| **unless** | trừ khi (= if...not) | **Unless** you practice, you won't improve. |
| **provided that** | với điều kiện là (trang trọng) | **Provided that** the test passes, we'll deploy. |
| **as long as** | miễn là, chừng nào | **As long as** you keep trying, you'll improve. |
| **in case** | phòng khi | Save your work **in case** the system crashes. |
| **otherwise** | nếu không thì | Study hard; **otherwise**, you'll fail the exam. |
| **on condition that** | với điều kiện là | I'll help you **on condition that** you help me later. |

## 2. TỪ CHỈ KẾT QUẢ / HỆ QUẢ

| Từ/Cụm | Ví dụ |
|---|---|
| **will** | If you practice, you **will** improve. |
| **can** | If you fix it, you **can** run the test. |
| **may/might** | If we try this, it **might** work. |
| **should** | If you're confused, you **should** ask. |
| **must** | If there's a bug, you **must** report it. |
| **then** | If A happens, **then** B happens. |

## 3. TỪ VỰNG QUAN TRỌNG — TÌNH HUỐNG VÀ HỆ QUẢ KỸ THUẬT

| Từ | Nghĩa | Ví dụ trong câu điều kiện |
|---|---|---|
| **trigger** | kích hoạt | If the sensor detects an obstacle, it will **trigger** an alert. |
| **activate** | kích hoạt, bật | If the button is pressed, it will **activate** the emergency stop. |
| **shut down** | tắt máy | If the temperature exceeds 80°C, the system will **shut down**. |
| **crash** | bị sập | If there's a memory leak, the program may **crash**. |
| **recover** | phục hồi | If we restart the node, the system will **recover**. |
| **exceed** | vượt quá | If the load **exceeds** capacity, the motor will overheat. |
| **detect** | phát hiện | If the LIDAR **detects** an obstacle, it will reroute. |
| **malfunction** | hoạt động sai | If the sensor **malfunctions**, the robot will stop. |
| **override** | ghi đè, ưu tiên | If you press E-stop, it will **override** all commands. |
