## 1. LIÊN TỪ KẾT HỢP (Coordinating Conjunctions — FANBOYS)

| Từ | Nghĩa | Ví dụ |
|---|---|---|
| **and** | và, cũng | Python is fast to write **and** easy to learn. |
| **but** | nhưng | C++ is powerful **but** difficult to master. |
| **or** | hoặc | We can fix it today **or** leave it for tomorrow. |
| **so** | vì vậy | The algorithm failed, **so** we tried another. |
| **yet** | nhưng (vẫn, mặc dù) | It's complex, **yet** the results are worth it. |
| **for** | vì (lý do) | We celebrated, **for** we achieved our goal. |
| **nor** | cũng không | He didn't try, **nor** did he give up gracefully. |

## 2. LIÊN TỪ PHỤ THUỘC (Subordinating Conjunctions — dùng trong câu phức)

| Từ | Nghĩa | Ví dụ |
|---|---|---|
| **although / even though** | mặc dù | **Although** it was difficult, we succeeded. |
| **because** | vì | The robot stopped **because** the battery died. |
| **when** | khi | I was coding **when** the power went out. |
| **while** | trong khi | She debugged **while** I wrote documentation. |
| **if** | nếu | **If** you study daily, you will improve quickly. |
| **unless** | trừ khi | **Unless** you practice, you won't improve. |
| **before** | trước khi | **Before** running the test, calibrate the sensors. |
| **after** | sau khi | **After** finishing the code, we ran the tests. |
| **since** | kể từ khi / vì | **Since** you're here, you can help me debug. |
| **until** | cho đến khi | I worked **until** the problem was solved. |
| **so that** | để mà | I took notes **so that** I could review later. |
| **as soon as** | ngay khi | **As soon as** I fix it, I'll send you the update. |

## 3. TRẠNG TỪ LIÊN KẾT (Conjunctive Adverbs — dùng sau ; hoặc đầu câu)

| Từ | Nghĩa | Ví dụ |
|---|---|---|
| **however** | tuy nhiên | The result was good. **However**, we can improve further. |
| **therefore** | vì vậy | The sensor failed; **therefore**, we stopped the test. |
| **moreover** | hơn nữa | The robot is fast. **Moreover**, it is very accurate. |
| **furthermore** | hơn nữa (trang trọng) | **Furthermore**, the cost is lower than expected. |
| **nevertheless** | dẫu vậy, tuy nhiên | It was difficult. **Nevertheless**, we didn't give up. |
| **consequently** | do đó, kết quả là | The update failed; **consequently**, the system crashed. |
| **in addition** | thêm vào đó | The robot is autonomous. **In addition**, it can work 8 hours. |
| **as a result** | kết quả là | We trained for months. **As a result**, we performed well. |
| **on the other hand** | mặt khác | Python is easy. **On the other hand**, C++ is faster. |
| **in contrast** | ngược lại | ROS 1 is stable. **In contrast**, ROS 2 offers more features. |

## 4. MẸO DÙNG ĐÚNG
- **FANBOYS** (and/but/or/so/yet/for/nor) → đi sau **dấu phẩy** trong câu ghép
- **however/therefore/moreover** → đi sau **dấu chấm phẩy (;)** hoặc đầu câu mới + **dấu phẩy (,)**
  - *The test passed; **however**, we found a minor issue.*
  - ***However**, we found a minor issue.*
