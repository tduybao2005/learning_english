## 2. TRẠNG TỪ TẦN SUẤT (Frequency Adverbs) — HOW OFTEN?

| Trạng từ | % xảy ra | Ví dụ |
|---|---|---|
| **always** | 100% | I **always** back up my code. |
| **usually / normally** | ~80% | He **usually** works until midnight. |
| **often / frequently** | ~60% | She **often** goes to the lab on weekends. |
| **sometimes / occasionally** | ~40% | I **sometimes** forget to push to GitHub. |
| **rarely / seldom** | ~20% | He **rarely** makes the same mistake twice. |
| **hardly ever** | ~5% | I **hardly ever** use Windows now. |
| **never** | 0% | She **never** gives up. |

## 3. TRẠNG TỪ MỨC ĐỘ (Degree Adverbs) — HOW MUCH?

| Trạng từ | Mức độ | Ví dụ |
|---|---|---|
| **extremely / incredibly** | cực kỳ | The code is **extremely** complex. |
| **very / really** | rất | She works **very** hard. |
| **quite / rather / fairly** | khá | It's **quite** a difficult problem. |
| **somewhat** | hơi, đôi chút | The result is **somewhat** disappointing. |
| **slightly** | hơi, một chút | The robot moved **slightly** to the left. |
| **hardly / barely / scarcely** | hầu như không | I **hardly** slept last night. |

## 4. TRẠNG TỪ LIÊN KẾT (Linking Adverbs) — dùng kết nối ý
- **however**: tuy nhiên — *The plan looks good. **However**, we need more data.*
- **therefore**: vì vậy — *The robot crashed. **Therefore**, we restarted the test.*
- **moreover / furthermore**: hơn nữa — ***Moreover**, the cost is too high.*
- **consequently**: do đó, kết quả là — ***Consequently**, we had to delay the demo.*
- **nevertheless**: tuy nhiên, dẫu vậy — ***Nevertheless**, we didn't give up.*
- **additionally**: thêm vào đó — ***Additionally**, we need more sensors.*
