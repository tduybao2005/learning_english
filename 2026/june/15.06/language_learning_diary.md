# Nhật ký học Tiếng Anh - Ngày 25 (15/06/2026)
*Cấu trúc "There is / There are" & Câu hỏi về sự tồn tại*

---

## PHẦN LÝ THUYẾT (30 phút)

### 1. Cấu trúc cơ bản

| Loại | Số ít | Số nhiều |
|---|---|---|
| **Khẳng định** | There **is** + N (ít) | There **are** + N (nhiều) |
| **Phủ định** | There **isn't** + N (ít) | There **aren't** + N (nhiều) |
| **Câu hỏi** | **Is there** + N (ít)? | **Are there** + N (nhiều)? |

*Ví dụ:*
- There **is** a sensor on the robot. (cảm biến)
- There **are** three bugs in the code. (ba lỗi)
- There **isn't** enough memory on the server. (không đủ bộ nhớ)
- **Is there** a manual for this device? (có sách hướng dẫn không?)
- **Are there** any errors in the output? (có lỗi nào không?)

### 2. Không nhầm với danh từ không đếm được
Danh từ không đếm được → dùng **is**:
- There **is** some data missing. (data = không đếm)
- There **is** a lot of information here.
- There **isn't** much time left.

### 3. Các thì khác nhau với "There is/are"

| Thì | Cấu trúc | Ví dụ |
|---|---|---|
| Hiện tại | There **is/are** | There **are** 10 sensors on the robot. |
| Quá khứ | There **was/were** | There **were** 5 errors last night. |
| Tương lai | There **will be** | There **will be** a meeting tomorrow. |
| Hiện tại hoàn thành | There **has/have been** | There **have been** many improvements. |

### 4. Câu hỏi và trả lời

| Câu hỏi | Trả lời ngắn |
|---|---|
| Is there a problem? | Yes, there **is**. / No, there **isn't**. |
| Are there any bugs? | Yes, there **are**. / No, there **aren't**. |
| How many sensors are there? | There **are** five sensors. |
| How much space is there? | There **is** about 50 GB of free space. |

### 5. Cách dùng "there" vs "it"
- **there** = để giới thiệu sự tồn tại của gì đó mới
- **it** = để nói về thứ đã đề cập rồi

*There **is** a new lab on campus. **It** is on the third floor of Building B.*

### 6. Hội thoại thực tế
> **A:** Is there a charging station in the lab?
> **B:** Yes, there is. There are actually three charging stations — one near the entrance and two at the back.
> **A:** Are there any robots available right now?
> **B:** There are two, but one is being repaired. So there's only one available.
> **A:** Was there a problem with the navigation system?
> **B:** Yes, there were several issues. There were problems with the path-planning algorithm. But there have been some updates since then.
> **A:** Is there enough battery for a 30-minute test?
> **B:** There should be. There was a full charge this morning.
