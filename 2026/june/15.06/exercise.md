## BÀI TẬP 1: Điền is/are/was/were/will be — 20 phút

1. There _______ a new lab on the third floor. (hiện tại)
2. There _______ three bugs in the code last night. (quá khứ)
3. There _______ a meeting tomorrow at 9 AM. (tương lai)
4. There _______ a lot of improvements since the last update. (present perfect: have/has been)
5. There _______ not enough time to finish the demo. (hiện tại)
6. There _______ only one robot available last week. (quá khứ)
7. There _______ some major changes in the next version. (tương lai)
8. _______ there any errors in the output? (hiện tại)
9. _______ there a problem during the test last time? (quá khứ)
10. How many sensors _______ there on the robot? (hiện tại)

**Đáp án:**
1. is | 2. were | 3. will be | 4. have been | 5. isn't | 6. was | 7. will be | 8. Are | 9. Was | 10. are

## BÀI TẬP 2: Chuyển sang phủ định và câu hỏi — 20 phút

**A. Phủ định:**
1. There is a problem with the algorithm.
2. There are several issues with the hardware.
3. There was enough data for the experiment.

**B. Câu hỏi Yes/No:**
4. There is a charging station in the lab.
5. There are two robots available.
6. There was a bug in the navigation code.

**C. Câu hỏi Wh-:**
7. There are 5 sensors on the robot. (How many?)
8. There is a meeting on Monday. (When?)

**Đáp án:**
1. There isn't a problem with the algorithm.
2. There aren't several issues with the hardware. / There aren't any issues with the hardware.
3. There wasn't enough data for the experiment.
4. Is there a charging station in the lab?
5. Are there two robots available?
6. Was there a bug in the navigation code?
7. How many sensors are there on the robot?
8. When is there a meeting?

6. There is no time to explain in detail — just fix the error.
7. There is a charging station at the end of the corridor, next to the storage room.

## BÀI TẬP 4: Đọc hiểu — 20 phút

> "In the university robotics lab, there are ten workstations equipped with computers, sensors, and electronic tools. There is a central testing area in the middle of the room where students can run robot demonstrations. At the back, there is a storage area for spare components and batteries.
>
> Currently, there are three robots in the lab: HospitGo, ArduBot, and NavBot. HospitGo is the most advanced — there are twelve different sensors mounted on it. There have been significant improvements to its navigation system since last semester.
>
> There was a power outage last Tuesday, which caused some data loss. As a result, there will be new backup procedures implemented next week. There is also a plan to install an additional UPS (uninterruptible power supply) system. This should ensure there are no further interruptions during critical tests."

**Câu hỏi:**
1. How many workstations are there?
2. Where is the testing area?
3. How many robots are there in the lab?
4. How many sensors are on HospitGo?
5. What happened last Tuesday?
6. What will happen next week?
7. Find 8 examples of "there is/are/was/were/will be/have been" in the text.

**Đáp án:**
1. Ten workstations.
2. In the middle of the room.
3. Three robots.
4. Twelve sensors.
5. There was a power outage, causing data loss.
6. New backup procedures will be implemented.
7. There are (ten workstations) / There is (a central testing area) / There is (a storage area) / There are (three robots) / There are (twelve sensors) / There have been (significant improvements) / There was (a power outage) / There will be (new backup procedures) / There is (a plan) / There are (no further interruptions)
