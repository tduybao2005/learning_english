# Nhật ký học Tiếng Anh - Ngày 26 (16/06/2026)
*Từ để hỏi mở rộng: How, Why, Which & Câu hỏi gián tiếp*

---

## PHẦN LÝ THUYẾT (30 phút)

### 1. Ôn lại tất cả Wh- Questions

| Từ hỏi | Hỏi về | Ví dụ |
|---|---|---|
| **What** | Cái gì / Làm gì | What **do** you study? |
| **Where** | Nơi chốn | Where **is** the lab? |
| **When** | Thời gian | When **does** the class start? |
| **Who** | Người (chủ ngữ) | Who **wrote** this code? |
| **Whom** | Người (tân ngữ) | Whom **did** you call? |
| **Why** | Lý do | Why **are** you late? |
| **How** | Cách thức | How **does** it work? |
| **Which** | Lựa chọn | Which **method** did you use? |
| **Whose** | Sở hữu của ai | Whose **laptop** is this? |

### 2. HOW + Adjective/Adverb — Hỏi về mức độ

| Cụm | Nghĩa | Ví dụ |
|---|---|---|
| **How long** | Bao lâu | How long **does** it take? |
| **How often** | Bao thường xuyên | How often **do** you study? |
| **How many** | Bao nhiêu (đếm được) | How many sensors **are** there? |
| **How much** | Bao nhiêu (không đếm/tiền) | How much **does** it cost? |
| **How far** | Bao xa | How far **is** the lab? |
| **How fast** | Nhanh như thế nào | How fast **can** the robot move? |
| **How old** | Bao nhiêu tuổi | How old **is** the professor? |
| **How well** | Tốt như thế nào | How well **do** you speak English? |

### 3. WHY — Hỏi lý do → Trả lời bằng BECAUSE

- *Why did the robot stop?* → *Because the battery died.*
- *Why are you learning English?* → *Because I want to take the IELTS exam.*

Cách trả lời khác lịch sự hơn:
- **(The reason) + is that + clause**: *The reason is that the sensor was broken.*
- **Due to + N**: *Due to a software error.*
- **As a result of + N**: *As a result of the power outage.*

### 4. WHICH — Câu hỏi lựa chọn
Dùng "which" khi có giới hạn lựa chọn (khác với "what"):
- *Which **method** did you use — gradient descent or backpropagation?* (có sẵn các lựa chọn)
- *What **language** do you use?* (không giới hạn lựa chọn)

### 5. Câu hỏi gián tiếp (Indirect Questions)
Trong văn lịch sự hoặc văn viết, dùng câu hỏi gián tiếp:

| Câu hỏi trực tiếp | Câu hỏi gián tiếp |
|---|---|
| Where **is** the lab? | Can you tell me **where the lab is**? |
| What **does** he do? | I wonder **what he does**. |
| Why **did** she leave? | Do you know **why she left**? |
| How **long does** it take? | Could you tell me **how long it takes**? |

*Chú ý:* Trong câu hỏi gián tiếp, trật tự từ là **S + V** (không đảo ngược như câu hỏi trực tiếp).

### 6. Hội thoại thực tế
> **A:** Could you tell me how the navigation system works?
> **B:** Sure. The robot uses LIDAR to map its environment and build a cost map. Then it uses path planning algorithms to find the optimal route.
> **A:** How fast can it move?
> **B:** It can reach up to 0.8 m/s, but we usually set it to 0.5 for safety.
> **A:** How long does it take to navigate from point A to B?
> **B:** It depends on the distance. It usually takes about 30 seconds to cross the corridor.
> **A:** Why did you choose ROS 2 instead of ROS 1?
> **B:** Because ROS 2 has better support for real-time systems and is more secure. Which path planning algorithm are you using? — We use the Nav2 stack with DWA.
