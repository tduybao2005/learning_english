## 1. CÁCH DIỄN ĐẠT LÝ DO (Expressing Reasons)

| Từ/Cụm | Dạng | Ví dụ |
|---|---|---|
| **because** | + mệnh đề | I'm late **because** the traffic was terrible. |
| **because of** | + danh từ | I'm late **because of** the traffic. |
| **due to** | + danh từ (trang trọng) | The delay was **due to** a software error. |
| **owing to** | + danh từ (trang trọng) | **Owing to** budget constraints, we delayed the project. |
| **as** | + mệnh đề (lý do đã biết) | **As** the battery was low, the robot stopped. |
| **since** | + mệnh đề (lý do đã biết) | **Since** I know the answer, I'll explain. |
| **the reason is that** | giải thích chính thức | **The reason is that** the sensor was miscalibrated. |
| **this is because** | giải thích trong văn viết | **This is because** AI requires large amounts of data. |

## 2. CÁCH DIỄN ĐẠT MỤC ĐÍCH (Expressing Purpose)

| Từ/Cụm | Dạng | Ví dụ |
|---|---|---|
| **to** (to-infinitive) | + V1 | I study **to** improve my English. |
| **in order to** | + V1 (trang trọng) | We designed the robot **in order to** help patients. |
| **so that** | + S + V (kết quả mong muốn) | I took notes **so that** I could review later. |
| **for the purpose of** | + V-ing (trang trọng) | This tool is used **for the purpose of** testing. |
| **with the aim of** | + V-ing (trang trọng) | She studied **with the aim of** getting a scholarship. |

## 3. CÁCH DIỄN ĐẠT CÁCH THỨC (Expressing Manner)

| Từ/Cụm | Ví dụ |
|---|---|
| **by + V-ing** | The robot learns **by analyzing** data from sensors. |
| **through + N/V-ing** | We improved the system **through** continuous testing. |
| **using + N** | The robot navigates **using** LIDAR and cameras. |
| **with the help of** | We completed the project **with the help of** our professor. |
| **in this way** | We fixed the bug. **In this way**, the robot worked correctly. |

## 4. COLLOCATIONS VỀ ĐẶT CÂU HỎI

| Collocation | Nghĩa |
|---|---|
| **raise a question** | đặt ra câu hỏi |
| **address a question** | trả lời/giải quyết câu hỏi |
| **pose a challenge** | đặt ra thách thức |
| **seek an answer** | tìm câu trả lời |
| **look into a problem** | tìm hiểu vấn đề |
| **investigate an issue** | điều tra vấn đề |
| **clarify a doubt** | làm rõ sự nghi ngờ |

## 5. CÁCH HỎI LỊCH SỰ (Polite Questioning)

| Thô lỗ/Trực tiếp | Lịch sự hơn |
|---|---|
| Where is the lab? | **Could you tell me** where the lab is? |
| What time does it start? | **Do you know** what time it starts? |
| Why did you do that? | **I was wondering** why you did that. |
| How much does it cost? | **Would you mind telling me** how much it costs? |
