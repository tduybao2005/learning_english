## BÀI TẬP 1: Phân loại đuôi -s — 25 phút

Phân loại các từ sau vào đúng cột /s/, /z/, hoặc /ɪz/:

*robots, files, buses, codes, tests, packages, errors, sensors, processes, modules, fixes, graphs, libraries, matches, algorithms, changes, works, runs, uses, concepts*

| /s/ | /z/ | /ɪz/ |
|---|---|---|
| | | |

**Đáp án:**
- **/s/**: robots, tests, graphs, concepts, works
- **/z/**: files, codes, errors, sensors, modules, algorithms, runs, libraries
- **/ɪz/**: buses, packages, processes, fixes, matches, changes, uses

## BÀI TẬP 4: Đọc hiểu & Nhận diện âm — 15 phút

Đọc đoạn văn sau. Gạch chân TẤT CẢ các từ có đuôi -s/-es và ghi cách phát âm (/s/, /z/, /ɪz/):

> "Bao develops autonomous navigation systems for hospital robots. Every day, he analyzes data from multiple sensors and tests different algorithms. The robot uses cameras and LIDAR to map its environment and processes thousands of data points per second.
>
> The system also detects obstacles and calculates the safest path. When the robot crashes into something, it stops and sends an error message. Bao fixes these issues by adjusting the algorithm parameters. His professor checks his progress weekly and always provides useful feedback."

**Đáp án:**
- **/ɪz/**: develops, analyzes, uses, processes, detects, calculates, crashes, fixes, adjusts, analyzes, processes
- **/s/**: maps, points, robots, tests, concepts, prompts
- **/z/**: sensors, algorithms, cameras, systems, thousands, parameters, issues, provides, stops, sends, errors

*(Chú ý: phân tích theo ÂM của chữ cuối, không phải chữ cái)*

## BÀI TẬP 5: Shadowing & Dictation — 10 phút

Đọc to 3 lần, chú ý đuôi -s:

1. He tests the sensors and fixes any errors he finds.
2. The system analyzes data and processes it in real time.
3. She publishes research papers and develops new algorithms.
4. The robot detects obstacles, calculates a path, and navigates safely.
5. The package includes three modules that work together.
6. He checks his code and runs the tests before every deployment.
7. The algorithm produces results, saves them to files, and generates a report.
8. She always uses tools that work efficiently and consume fewer resources.
