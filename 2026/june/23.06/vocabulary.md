## PHÂN BIỆT DỄ NHẦM — ÔN LẠI TRƯỚC KHI LÀM BÀI

| Điểm nhầm | Sai | Đúng |
|---|---|---|
| Although + but | Although it was hard, **but** I passed. | Although it was hard, I passed. |
| Despite + mệnh đề | Despite **it was raining** | Despite **the rain** |
| Since / For | I've studied since **2 months** | I've studied **for** 2 months |
| For / Since | I've studied for **May** | I've studied **since** May |
| Ever/Never + thì | Did you **ever** visit? | **Have** you **ever** visited? |
| Was/Were + V-ing | He was work hard | He **was working** hard |
| Going to + have | She's going to **has** | She's going to **have** |
| There is + plural | There **is** many robots | There **are** many robots |
| State verbs + -ing | I **am knowing** | I **know** |
| Do/Does + V | She **does speaks** | She **speaks** |
