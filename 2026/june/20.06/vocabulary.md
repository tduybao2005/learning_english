# TỪ VỰNG TIẾNG ANH - NGÀY 30 (20/06/2026)

## 1. TỪ CHỈ THỜI GIAN QUÁ KHỨ (Simple Past)

| Từ/Cụm | Nghĩa | Ví dụ |
|---|---|---|
| **yesterday** | hôm qua | I fixed the bug **yesterday**. |
| **last night/week/month/year** | tối qua / tuần/tháng/năm ngoái | I trained **last month**. |
| **... ago** | ... trước | I started IELTS prep two months **ago**. |
| **in + year** | vào năm | She graduated **in** 2023. |
| **when + past clause** | khi... | **When** I was a first-year student, ... |
| **at** + specific time | lúc | The power went out **at** 11 PM. |

## 2. TỪ CHỈ THỜI GIAN HIỆN TẠI HOÀN THÀNH (Present Perfect)

| Từ | Vị trí | Ví dụ |
|---|---|---|
| **just** | have/has + just + V3 | I have **just** finished the code. |
| **already** | have/has + already + V3 | She has **already** submitted. |
| **yet** | Cuối câu (hỏi/phủ) | Have you tested it **yet**? / Not **yet**. |
| **ever** | Câu hỏi: have + ever + V3 | Have you **ever** used ROS? |
| **never** | have/has + never + V3 | I have **never** failed a test before. |
| **recently** | Cuối hoặc đầu câu | I have **recently** started using nvim. |
| **so far** | Cuối câu | I have fixed 5 bugs **so far**. |
| **since** | since + time point | I've studied English **since** May. |
| **for** | for + duration | I've been here **for** two months. |

## 4. CỤM ĐỘNG TỪ VỀ DỰ ÁN / KỸ THUẬT (quá khứ)

| Cụm từ | Nghĩa | Ví dụ |
|---|---|---|
| **carry out / conduct** | thực hiện | We **carried out** several tests last week. |
| **encounter** | gặp phải | We **encountered** a serious error. |
| **resolve / overcome** | giải quyết / vượt qua | They **resolved** the issue in 2 days. |
| **deploy** | triển khai | We **deployed** the system last Friday. |
| **implement** | thực thi, áp dụng | She **implemented** the new algorithm. |
| **achieve** | đạt được | The team **achieved** its target. |
| **collaborate** | cộng tác | We **collaborated** with another team. |
| **present** | trình bày | I **presented** the project at the fair. |
