## 1. DANH TỪ TẬP THỂ (Collective Nouns) — số ÍT trong văn viết

| Danh từ | Nghĩa | Ví dụ |
|---|---|---|
| **team** | đội nhóm | The **team** is ready. |
| **group** | nhóm | The **group** has finished the task. |
| **committee** | ban, ủy ban | The **committee** has voted. |
| **staff** | nhân viên (toàn bộ) | The **staff** is well-trained. |
| **class** | lớp học | The **class** is very motivated. |
| **audience** | khán giả | The **audience** was impressed. |
| **crew** | tổ, đội (kỹ thuật) | The **crew** is fixing the issue. |
| **faculty** | khoa, giảng viên | The **faculty** has approved the project. |
| **board** | hội đồng | The **board** is meeting tomorrow. |
| **jury** | ban giám khảo | The **jury** has made its decision. |

## 2. TỪ CHỈ SỐ LƯỢNG (Quantifiers)

| Từ | Đi với | Nghĩa | Ví dụ |
|---|---|---|---|
| **all** | C số nhiều / U | tất cả | **All** the sensors are working. |
| **most** | C số nhiều / U | hầu hết | **Most** of the code is clean. |
| **some** | C số nhiều / U | một số | **Some** robots are autonomous. |
| **any** | C số nhiều / U | bất kỳ | Is there **any** feedback? |
| **few** | C số nhiều | ít (không đủ) | **Few** students understand this. |
| **a few** | C số nhiều | một vài (đủ dùng) | I have **a few** suggestions. |
| **little** | U | ít (không đủ) | There is **little** time left. |
| **a little** | U | một chút (đủ dùng) | I need **a little** help. |
| **many** | C số nhiều | nhiều (đếm được) | **Many** bugs were found. |
| **much** | U | nhiều (không đếm) | It takes too **much** time. |
| **a lot of** | C và U | nhiều | **A lot of** data is needed. |
| **plenty of** | C và U | rất nhiều, dư đủ | There's **plenty of** time. |
| **enough** | C và U | đủ | We have **enough** sensors. |
| **no** | C và U | không có | There are **no** errors. |

## 3. PHÂN BIỆT few/a few và little/a little

| | Nghĩa | Ý nghĩa cảm xúc |
|---|---|---|
| **few** | ít (đếm được) | Tiêu cực — không đủ |
| **a few** | một vài (đếm được) | Tích cực — đủ dùng |
| **little** | ít (không đếm được) | Tiêu cực — không đủ |
| **a little** | một chút (không đếm được) | Tích cực — đủ dùng |

*Ví dụ:*
- **Few** students passed. (= Rất ít sinh viên — kết quả xấu)
- **A few** students passed. (= Một vài sinh viên — có hy vọng)
- There is **little** data. (= Quá ít dữ liệu — vấn đề)
- There is **a little** data. (= Có một chút dữ liệu — dùng tạm được)

## 4. COLLOCATION VỀ SỐ LƯỢNG
- **the majority of**: đa số — *The majority of robots use cameras.*
- **the minority of**: thiểu số
- **a large number of**: số lượng lớn (đếm được)
- **a large amount of**: số lượng lớn (không đếm được)
- **a wide range of**: đa dạng
- **the vast majority**: tuyệt đại đa số
