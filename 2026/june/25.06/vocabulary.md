## 1. ĐỘNG TỪ V1 / V3 THƯỜNG DÙNG TRONG BỊ ĐỘNG

| V1 | V3 | Nghĩa | Ví dụ bị động |
|---|---|---|---|
| **design** | designed | thiết kế | The robot was **designed** to help nurses. |
| **build** | built | xây dựng | The prototype was **built** in 3 months. |
| **test** | tested | kiểm tra | The system is **tested** every week. |
| **develop** | developed | phát triển | A new algorithm has been **developed**. |
| **install** | installed | cài đặt | The sensors were **installed** correctly. |
| **calibrate** | calibrated | hiệu chỉnh | All sensors must be **calibrated** before use. |
| **implement** | implemented | triển khai | The new feature was **implemented** last week. |
| **optimize** | optimized | tối ưu hóa | The code has been **optimized** for speed. |
| **integrate** | integrated | tích hợp | The modules were **integrated** into one system. |
| **analyze** | analyzed | phân tích | The data were **analyzed** using Python. |
| **configure** | configured | cấu hình | The network was **configured** by the admin. |
| **update** | updated | cập nhật | The software is **updated** every month. |
| **connect** | connected | kết nối | The devices were **connected** via Bluetooth. |
| **power** | powered | cấp điện | The robot is **powered** by a lithium battery. |
| **repair** | repaired | sửa chữa | The broken sensor is being **repaired** now. |

## 2. TỪ VỰNG VĂN KHOA HỌC — BÁO CÁO KỸ THUẬT

| Cụm từ | Nghĩa | Ví dụ |
|---|---|---|
| **was/were conducted** | được thực hiện | The experiment **was conducted** in the lab. |
| **was/were collected** | được thu thập | Data **were collected** over six months. |
| **was/were observed** | được quan sát | Significant improvement **was observed**. |
| **was/were found** | được tìm thấy | An error **was found** in the algorithm. |
| **was/were demonstrated** | được chứng minh | The concept **was demonstrated** successfully. |
| **have been made** | đã được thực hiện | Several improvements **have been made**. |
| **is/are expected to** | được kỳ vọng | The system **is expected to** launch in 2027. |
| **it is believed that** | người ta tin rằng | **It is believed that** AI will transform medicine. |
| **it is estimated that** | ước tính rằng | **It is estimated that** 20% of jobs will be automated. |
| **it has been shown that** | đã được chứng minh | **It has been shown that** robots reduce errors. |

## 3. BỊ ĐỘNG TRONG IELTS WRITING TASK 1 (Mô tả biểu đồ)

| Active (ít dùng) | Passive (nên dùng) |
|---|---|
| People produced more cars. | More cars **were produced**. |
| The data shows... | It **is shown** that... |
| Someone collected the data in 2020. | The data **were collected** in 2020. |
| They found that AI improved accuracy. | It **was found** that AI improved accuracy. |
