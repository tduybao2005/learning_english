# Nhật ký học Tiếng Anh - Ngày 35 (25/06/2026)
*Câu bị động — Passive Voice (Introduction)*

---

## PHẦN LÝ THUYẾT (30 phút)

### 1. Câu chủ động vs Câu bị động

| | Câu chủ động (Active) | Câu bị động (Passive) |
|---|---|---|
| **Tiêu điểm** | Người/vật thực hiện hành động | Người/vật nhận hành động |
| **Ví dụ** | The engineer **built** the robot. | The robot **was built** by the engineer. |

*Khi nào dùng bị động?*
1. Không biết hoặc không quan trọng ai thực hiện: *The lab **was cleaned** last night.*
2. Trong văn khoa học/báo cáo (trang trọng): *The data **were analyzed** using Python.*
3. Nhấn mạnh đối tượng bị tác động: *HospitGo **was designed** to assist hospital staff.*

### 2. Cấu trúc bị động các thì

| Thì | Chủ động | Bị động | Cấu trúc bị động |
|---|---|---|---|
| Simple Present | They **test** the robot. | The robot **is tested**. | am/is/are + V3 |
| Present Continuous | They **are testing** the robot. | The robot **is being tested**. | am/is/are + being + V3 |
| Simple Past | They **tested** the robot. | The robot **was tested**. | was/were + V3 |
| Past Continuous | They **were testing** it. | It **was being tested**. | was/were + being + V3 |
| Present Perfect | They **have tested** the robot. | The robot **has been tested**. | have/has + been + V3 |
| Simple Future | They **will test** the robot. | The robot **will be tested**. | will + be + V3 |
| Modal | They **must test** the robot. | The robot **must be tested**. | modal + be + V3 |

### 3. Cách chuyển câu chủ động → bị động

**Bước 1:** Tân ngữ (object) của câu chủ động → Chủ ngữ (subject) của câu bị động
**Bước 2:** Chia "to be" theo thì và số
**Bước 3:** Động từ chính → dạng V3 (past participle)
**Bước 4:** (Tùy chọn) Thêm "by + chủ ngữ cũ"

*Ví dụ:*
- Active: The team **designed** the robot.
  → S = "The team", V = "designed" (SP), O = "the robot"
- Passive: The robot **was designed** (by the team).
  → S = "The robot", V = "was designed" (SP passive), by-agent = "(by the team)"

### 4. Bị động với Modal Verbs

| Modal | Bị động |
|---|---|
| can | can **be** + V3 |
| should | should **be** + V3 |
| must | must **be** + V3 |
| have to | have to **be** + V3 |
| may/might | may/might **be** + V3 |

*Ví dụ:*
- Sensors **must be calibrated** before each test.
- The data **should be backed up** every day.
- The bug **may be fixed** in the next update.

### 5. Bị động trong văn khoa học/báo cáo (Academic Passive)

Rất phổ biến trong báo cáo kỹ thuật và IELTS Writing Task 1:
- *The experiment **was conducted** over three months.*
- *Data **were collected** from 50 participants.*
- *The results **were analyzed** using statistical software.*
- *A prototype **has been developed** to test the concept.*
- *Several improvements **have been made** to the original design.*

### 6. Hội thoại thực tế
> **A:** When was HospitGo first built?
> **B:** The first prototype was designed in September 2025. It was built by a team of four students.
> **A:** Has it been tested in a real hospital yet?
> **B:** Not yet. It's currently being tested in a simulated environment. All sensors have been calibrated and the navigation algorithm has been optimized.
> **A:** What improvements have been made recently?
> **B:** The path-planning algorithm was completely rewritten last month. The delivery speed has been increased by 30%, and a new obstacle avoidance system has been integrated.
