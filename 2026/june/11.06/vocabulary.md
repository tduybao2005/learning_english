## 2. ACADEMIC WORD LIST (AWL) — Nhóm 1 (Học thuật cần biết cho IELTS)

| Từ | Nghĩa | Ví dụ |
|---|---|---|
| **analyze** | phân tích | We need to **analyze** the data. |
| **approach** | cách tiếp cận / tiếp cận | A new **approach** to problem-solving. |
| **assess** | đánh giá | The professor will **assess** our project. |
| **concept** | khái niệm | This is a fundamental **concept**. |
| **demonstrate** | chứng minh, trình diễn | She will **demonstrate** the algorithm. |
| **evaluate** | đánh giá (kết quả) | We need to **evaluate** the results. |
| **identify** | xác định | Can you **identify** the problem? |
| **implement** | thực hiện | We plan to **implement** this solution. |
| **indicate** | chỉ ra | The data **indicates** a clear trend. |
| **obtain** | đạt được, thu được | We **obtained** promising results. |
| **require** | yêu cầu | The project **requires** teamwork. |
| **significant** | đáng kể, quan trọng | This is a **significant** achievement. |
| **specific** | cụ thể | Please be more **specific**. |
| **structure** | cấu trúc | The **structure** of the essay is clear. |
