| V1 | V2 | R/I |
|---|---|---|
| study | | |
| go | | |
| fix | | |
| write | | |
| upload | | |
| build | | |
| want | | |

| find | | |
| configure | | |
| make | | |
| spend | | |

**Đáp án:**
study → **studied** (R) | go → **went** (I) | fix → **fixed** (R) | write → **wrote** (I) | upload → **uploaded** (R) | build → **built** (I) | want → **wanted** (R) | send → **sent** (I) | test → **tested** (R) | find → **found** (I) | configure → **configured** (R) | make → **made** (I) | spend → **spent** (I) | try → **tried** (R) | feel → **felt** (I)

## BÀI TẬP 3: Phát âm đuôi -ed — Phân loại — 15 phút

Phân loại các từ sau vào ba cột /t/, /d/, /ɪd/:

*worked, needed, studied, stopped, opened, wanted, fixed, downloaded, tested, finished, used, started, watched, configured, waited*

| /t/ | /d/ | /ɪd/ |
|---|---|---|
| | | |

**Đáp án:**
- **/t/**: worked, stopped, fixed, finished, watched
- **/d/**: studied, opened, downloaded, used, configured
- **/ɪd/**: needed, wanted, tested, started, waited

## BÀI TẬP 4: Dịch Việt → Anh — 20 phút

1. Tôi đã học 4 tiếng đồng hồ tối qua.
2. Cô ấy đã tìm thấy giải pháp cho lỗi đó.
3. Chúng tôi đã xây dựng robot đó trong 3 tháng.
4. Anh ấy đã gửi báo cáo cho giáo sư vào sáng nay.
5. Tôi đã tiêu hết ngày Thứ Bảy để viết code.

4. He sent the report to the professor this morning.
5. I spent all Saturday writing code.
6. They tested the sensors in the lab.
7. I felt very proud when the robot worked.
8. He bought a new sensor at the electronics store.

## BÀI TẬP 5: Đọc hiểu — 15 phút

Đọc đoạn văn và trả lời câu hỏi:

> "Last semester was a very productive period for our robotics team. We started the project in February with only a basic idea. In the first month, we built the chassis and configured the motors. Then we wrote the control software in Python. We used ROS 2 for the navigation system.
>
> We faced many challenges. We broke two sensors during testing and spent a lot of money buying replacements. We also found a serious bug in the pathfinding algorithm. It took us two weeks to fix it.
>
> Finally, in May, we sent our project report to the faculty. We felt very satisfied because we made significant progress. The robot went from a simple concept to a working prototype."

**Câu hỏi:**
1. When did the team start the project?
2. What did they build in the first month?
3. What language did they write the software in?
4. What problems did they face?
5. How long did it take to fix the bug?
6. What did they send to the faculty?
7. How did they feel at the end?

**Đáp án:**
1. They started in February.
2. They built the chassis and configured the motors.
3. They wrote it in Python.
4. They broke two sensors and found a serious bug in the pathfinding algorithm.
5. It took two weeks.
6. They sent their project report.
7. They felt very satisfied.
