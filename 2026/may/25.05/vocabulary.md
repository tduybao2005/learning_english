## 3. CỤM ĐỘNG TỪ ĐI KÈM (Common Collocations — Simple Past)
- **spent** time/money on: *I spent 6 hours on the project.*
- **made** a mistake / **made** progress: *I made a mistake in the code.*
- **wrote** a report / code / email: *She wrote a detailed report.*
- **found** a bug / solution / error: *We found a critical bug yesterday.*
- **built** a model / robot / circuit: *They built a prototype last month.*
- **sent** an email / message / file: *I sent the report to the professor.*

## 4. MẸO NHỚ IRREGULAR VERBS (Memory Tricks)
- **go → went**: hoàn toàn khác — nhớ riêng!
- **buy, bring, think** → đều biến thành **-ought**: bought, brought, thought
- **send, spend, lend, bend** → đều biến thành **-ent**: sent, spent, lent, bent
- **keep, sleep, feel, kneel** → đều biến thành **-ept**: kept, slept, felt (ngoại lệ)
- **build, find, hold, bind** → đều biến thành **-ilt/-ound**: built, found, held, bound
