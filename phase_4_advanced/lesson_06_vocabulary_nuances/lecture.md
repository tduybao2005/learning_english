# BÀI 6: SẮC THÁI TỪ NGỮ — VOCABULARY NUANCES

## Tại sao phải học Vocabulary Nuances?

Từ đồng nghĩa (synonyms) không bao giờ hoàn toàn giống nhau. Mỗi từ mang một **sắc thái (nuance)** riêng về:
- Mức độ cảm xúc (emotional strength)
- Register (formal/informal/neutral)
- Connotation tích cực/tiêu cực/trung lập
- Mức độ chắc chắn (certainty)
- Frequency/collocation (đi với từ gì)

---

## PHẦN 1: CẶP TỪ ĐỒNG NGHĨA CÓ SẮC THÁI KHÁC NHAU

### 1.1 Mô tả tính cách — Positive vs Negative Framing
| Positive (tích cực) | Negative (tiêu cực) | Nghĩa trung lập |
|---|---|---|
| **steadfast** (vững vàng) | **obstinate** (ngoan cố) | kiên định |
| **thrifty** (tiết kiệm khôn ngoan) | **miserly** (bủn xỉn) | tiêu ít tiền |
| **confident** (tự tin) | **arrogant** (kiêu ngạo) | tin vào bản thân |
| **assertive** (quyết đoán) | **aggressive** (hung hăng) | nói thẳng ý kiến |
| **inquisitive** (tò mò học hỏi) | **nosy** (tò mò soi mói) | muốn biết chuyện |
| **meticulous** (tỉ mỉ cẩn thận) | **finicky** (khó tính quá mức) | chú ý từng chi tiết |
| **economical** (tiết kiệm hợp lý) | **cheap** (rẻ tiền, ki bo) | tránh lãng phí |
| **determined** (quyết tâm) | **stubborn** (bướng bỉnh) | không từ bỏ |
| **idealistic** (có lý tưởng) | **naive** (ngây thơ) | tin vào điều tốt đẹp |
| **slim** (mảnh mai) | **scrawny** (gầy gò, xơ xác) | không nặng |

**Ví dụ về spin (cách đóng khung):**
- *She is **determined** to succeed.* → tích cực
- *She is **stubborn** and refuses to listen.* → tiêu cực
- (Cùng mô tả một tính cách nhưng từ hai góc nhìn khác nhau)

### 1.2 Cảm xúc — Mức độ cường độ
**Hạnh phúc (từ nhẹ đến mạnh):**
content → pleased → happy → delighted → ecstatic → euphoric

**Buồn (từ nhẹ đến mạnh):**
sad → unhappy → upset → distressed → devastated → inconsolable

**Tức giận (từ nhẹ đến mạnh):**
irritated → annoyed → frustrated → angry → furious → enraged → livid

**Sợ hãi (từ nhẹ đến mạnh):**
uneasy → nervous → anxious → worried → frightened → terrified → petrified

**Ngạc nhiên (từ nhẹ đến mạnh):**
surprised → astonished → astounded → flabbergasted

### 1.5 Formal vs Informal register
| Informal | Formal/Academic | Nghĩa |
|---|---|---|
| get | obtain / acquire / receive | nhận được |
| use | utilise / employ / apply | sử dụng |
| show | demonstrate / illustrate / indicate | cho thấy |
| think | consider / believe / argue | suy nghĩ |
| look at | examine / analyse / investigate | xem xét |
| make | produce / generate / create | tạo ra |
| big | significant / substantial / considerable | lớn |
| small | minimal / negligible / marginal | nhỏ |
| good | beneficial / advantageous / favourable | tốt |
| bad | detrimental / harmful / adverse | xấu |
| need | require / necessitate | cần |
| find out | discover / establish / reveal | khám phá |

### 1.6 Mức độ chắc chắn — Hedging in Academic Writing
**Chắc chắn tuyệt đối (100%):**
definitely / certainly / undoubtedly / clearly / obviously

**Khá chắc chắn (80-90%):**
probably / likely / generally / typically / tends to

**Có thể (50-70%):**
possibly / perhaps / may / might / could / it is possible that

**Nhẹ hơn (30-50%):**
apparently / seemingly / it appears that / it seems that / it would seem

**Quan điểm người khác (trích dẫn):**
it is claimed that / it has been argued that / according to / as X suggests

## Tóm tắt — Cách sử dụng Nuance trong IELTS

**Writing Task 2:** Chọn từ dựa trên connotation phù hợp với lập luận
- *Stubborn* (tiêu cực) vs *determined* (tích cực) — biết cách "spin" để phục vụ luận điểm

**Speaking:** Dùng intensity gradients để nghe tự nhiên hơn
- Thay *very happy* → *delighted* / *thrilled* / *overjoyed*
- Thay *very sad* → *devastated* / *heartbroken*

**Register awareness:** Biết khi nào dùng formal vs informal
- Báo cáo IELTS Task 2: *"This is detrimental to..."* KHÔNG phải *"This is bad for..."*
